## 初始阶段

先安装一个Linux系统，推荐用WSL。

WSL不太占用系统空间，在Windows上直接安装与Windows也是隔离的，虚拟机要重装一个系统，系统空间占用率较高，文件共享和网络配置也麻烦，如果双系统的话会更麻烦，还容易搞坏磁盘分区。

学习命令行的基础操作，`cd` `ls` `mkdir` `rm` 这种的文件操作

会使用一些工具， `cat` `vim` 这些文件查看工具，以及网络请求 `curl` `wget` 这些网络请求工具的使用，磁盘管理，进程和线程管理，（简单的能看懂就行）

权限和用户的概念，如何查看权限，如何修改权限，如何配置最高权限

shell脚本的使用和语法学习。能看懂环境变量的的配置和意思，

## 有感觉后

学习 `GCC` `GBD` `Cmake` 这些C/C++的编译方法和项目管理，以及调试方法，还有项目交叉编译。
