## RL的基础

- 智能体（Agent）
- 环境 (Environment)
- 状态 (State, s)
- 动作 (Action, a)
- 奖励 (Reward, r)
- 策略 (Policy, π)
- 回报 (Return, G)
- 价值函数 (Value Function)
- 马尔可夫决策过程 (MDP)

### 核心算法

贝尔曼方程：贝尔曼方程是所有 RL 算法的推导基础，核心逻辑是：**当前状态的价值 = 即时奖励 + 未来所有状态的折扣价值的期望**，它把当前决策和长期收益关联起来，构建了 RL 的优化闭环。
