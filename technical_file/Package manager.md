## 软件包的来历

在以前开源软件只提供了.tar.gz的源码文件，用户必须自己编译每个想在GNU/Linux上运行的软件，当Debian诞生时，这样的一个管理工具dpkg也就应运而生，可以用来管理deb后缀的“包“文件。从而著名的“package“概念第一次出现在GNU/Linux系统中。



## 什么是包管理器

包管理器就像是手机上的应用商店一样。当我们需要往电脑上安装应用的时候就可以从它哪里去下载我们需要的应用。

不同的Linux发行版也有不同的包管理器

### APT

apt update命令：更新软件包列表   相当于把一份最新的软件包列表下载到我们本地







debian系统包管理：dpkg  **apt**
arch系统包管理：pacman
gentoo系统包管理：emerge
slackware包管理：pkg
solus包管理：eopkg
redhat系统包管理：**dnf yum**
SUSE包管理：zypper
